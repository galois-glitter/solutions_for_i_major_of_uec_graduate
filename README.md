## 解答作成の手順

まず、対応する年度のディレクトリを作成する。（例：`2023`）

### 必須問題

`_required.tex`を作成する年度のディレクトリにコピーする。そのとき、ファイル名の先頭に年度の数をいれる。（例：`2023/2023_required.tex`）

### 選択問題

`_mcq.tex`を作成する年度のディレクトリにコピーする。そのとき、ファイル名の先頭に年度の数をいれる。（例：`2023/2023_mcq.tex`）

### 解答作成ルール

* `\titlepage`は使用しない
* 大問ごとに`\newpage`する
